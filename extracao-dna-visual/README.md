# Extração de DNA Visual

Esta pasta contém os resultados da análise estrutural das miniaturas utilizando o framework **VDNA-TEMPLATE™**.

O objetivo desta etapa foi decompor as thumbnails em camadas estruturais.

---

# Estrutura analisada

A análise considera múltiplos níveis visuais:

### Identidade visual macro

- paleta de cores
- contraste
- atmosfera visual
- temperatura de cor

### Composição

- posição do sujeito
- escala do personagem
- hierarquia visual
- profundidade

### Textura

- nitidez
- partículas
- haze ou fumaça
- iluminação

### Arquétipos visuais

- autoridade
- ameaça
- proteção
- revelação

---

# Output da análise

O resultado da extração é representado em formato JSON.

Esse formato permite:

- padronizar análises
- comparar diferentes thumbnails
- gerar templates visuais

O arquivo `exemplo_vdna_output.json` apresenta um exemplo dessa estrutura.
