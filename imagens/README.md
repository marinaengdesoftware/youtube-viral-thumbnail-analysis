# Imagens do Estudo

Esta pasta contém exemplos visuais utilizados durante o estudo.

Estrutura:

- `thumbs_original` → miniaturas coletadas do canal analisado
- `thumbs_recriadas` → thumbnails geradas a partir do template extraído

---

# Objetivo

Permitir comparação entre:

- thumbnails originais
- thumbnails recriadas com base no template

Isso demonstra como padrões visuais podem ser transformados em estruturas replicáveis.
